# Model Card: Diabetes Risk Assessment

## Summary

This model estimates diabetes risk from eight structured clinical inputs. It is built for engineering review and local experimentation. It has not been validated for clinical use.

## Model Details

- **Version**: 1.1
- **Date**: 2026-05-07
- **Model**: XGBoost classifier with sigmoid calibration
- **Pipeline**: sklearn imputation, feature engineering, scaling, classifier, calibration
- **Artifact**: `models/clinical_pipeline.joblib`
- **Training command**: `python -m src.models.train`

## Intended Use

Use this model to inspect the ML pipeline, API contract, validation rules, and monitoring hooks in this repository.

Do not use it for diagnosis, care decisions, patient triage, or any workflow where a person could be harmed by an incorrect prediction.

## Inputs

| Feature | Type | Accepted range |
| --- | --- | --- |
| Pregnancies | int | 0 to 20 |
| Glucose | float | 0 to 500 |
| BloodPressure | float | 0 to 250 |
| SkinThickness | float | 0 to 120 |
| Insulin | float | 0 to 900 |
| BMI | float | 0 to 80 |
| DiabetesPedigreeFunction | float | 0 to 3 |
| Age | int | 0 to 120 |

The API rejects extra fields, booleans, NaN, infinity, and out-of-range values.

## Data

The default dataset is generated by `src.data.generator`. It is shaped like a diabetes risk dataset but does not represent a real clinical cohort.

## Known Limitations

- No external validation.
- No subgroup performance report in the standard training output.
- No uncertainty intervals around metrics.
- No clinical calibration review.
- No prospective testing.

## Risk Notes

- False negatives could delay follow-up if the model were misused.
- False positives could create unnecessary concern.
- SHAP values can help inspect model behavior, but they do not establish medical causality.
- Model behavior may change on populations that differ from the synthetic training data.

## Operational Constraint

Only load model artifacts from trusted sources. `joblib` is not a safe interchange format for untrusted files.
